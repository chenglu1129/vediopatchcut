# 批量视频生成工具

基于 Python + tkinter + ffmpeg 的桌面工具，自动将素材视频随机拼接并配上音频，批量输出指定数量的成品视频。

## 功能特性

- 从素材文件夹随机抽取片段，自动拼接至音频总时长
- 支持设置备选片头（多个片头等权重随机抽取）
- 支持竖屏 9:16 / 横屏 16:9 两种画幅
- 一键预处理素材（统一转码为目标分辨率 + 30fps）
- 提取视频中的音频轨道为 MP3
- 配置可保存/加载，支持多套配置命名管理
- 生成进度实时显示，支持随时取消

## 环境要求

- Python 3.8+
- ffmpeg（放在程序目录下的 ffmpeg/ 文件夹，或系统 PATH 中）
- 依赖库：sv-ttk

安装依赖：

    pip install sv-ttk

## 使用方法

    python app.py

### 操作步骤

1. 素材文件夹：选择包含 MP4 视频片段的目录
2. 备选片头（可选）：添加片头视频，每条成品随机抽取一个拼在最前面
3. 音频文件：选择配乐 MP3/WAV，成品时长以此为准
4. 视频画幅：竖屏（1080x1920）或横屏（1920x1080）
5. 生成数量：设置批量生成的视频条数
6. 输出目录：选择成品保存位置
7. 点击开始生成

### 预处理素材（推荐）

首次使用前点击预处理，将素材统一转码为目标分辨率和帧率。原始文件备份至素材目录下的 _backup/ 子文件夹。

### 提取音频

点击提取音频按钮，可从视频文件中提取 MP3 音轨。

## 打包为可执行文件

    pip install pyinstaller
    pyinstaller vediopatchcut.spec

打包后程序位于 dist/ 目录，需将 ffmpeg/ 文件夹放在同级目录下。

## 目录结构

    vediopatchcut/
    ├── app.py          # 主程序
    ├── ffmpeg/         # ffmpeg 二进制（不含在仓库中）
    ├── config.json     # 上次配置（自动生成，不提交）
    └── *.json          # 手动保存的命名配置
